# X-UI Self

I built **[X-UI Self](https://x-ui-self.vercel.app/)** as a dashboard template I’d actually want to use—minimal, responsive, dark mode by default. No bloat; just a solid base for admin panels, MVPs, or internal tools. If you’ve ever said “I’ll fix the UI later” and never did, this is the kind of starting point that makes it easier to ship something that looks good from day one.

> Your dashboard called. It wants to look like _this_ now.

---

## What’s in it

- **Next.js + Tailwind** — Fast, familiar stack.
- **Lucide Icons** — Clean, consistent icons.
- **Responsive** — Works from phone to big monitors.
- **Dark mode** — Easy on the eyes.
- **ShadCN UI** — Good components without reinventing the wheel.

---

## Who it’s for

Devs building SaaS or internal tools, founders shipping MVPs, or anyone who wants a dashboard that doesn’t look like a default Bootstrap theme.

---

## Tech

Next.js 14, TailwindCSS 3.x, TypeScript, Lucide React, ShadCN UI.
