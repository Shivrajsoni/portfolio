##  Example 

Imagine you're out for lunch with a friend, and you need to split the bill.

1.  **Adding a Contact:** You've previously added your friend, Alex, to your Paysol contacts by saving his username and public key.
2.  **Making a Payment:** To pay him back, you open Paysol, select "Alex" from your contact list, enter the amount, and hit "Send".
3.  **Instant Confirmation:** Erite instantly processes the transaction on the Solana network. You get a confirmation toast, and Alex receives the funds in his wallet.

No more fumbling with long addresses or worrying about sending funds to the wrong place. It's that simple!

## ğŸ Getting Started

Want to run Erite locally or contribute to the project? Follow these steps:

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/your-username/paysol.git
    cd paysol
    ```

2.  **Install Dependencies**
    We use `bun` for package management.
    ```bash
    bun install
    ```

3.  **Set Up Environment Variables**
    Create a `.env` file in the root of the project and populate it with the necessary keys (e.g., database URL, Clerk credentials).
    ```bash
    cp .env.example .env
    ```

4.  **Push Database Schema**
    Apply the database schema using Prisma.
    ```bash
    bunx prisma db push
    ```

5.  **Run the Development Server**
    ```bash
    bun dev
    ```
    Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
